## Code families and definitions

| Code | Definition | Example indicators |
|------|-----------|-------------------|

| dignity_autonomy | Preservation of dignity, self-worth, independence, and decision-making agency — especially for elderly | كرامة, كبار القدر, استقلالية, "not a burden", "حفظ كرامة المريض", "حق القرار" |
| inner_peace_stability | Internal tranquility, psychological rest, stability, and freedom from inner turmoil | راحة نفسية, استقرار, سكينة, طمأنينة, "peace of mind", "calm" |
| safety_security | Physical and psychological safety, protection, and stability as preconditions for wellbeing | أمن, أمان, حماية, استقرار, "safety", "secure" |

## Codebook notes
- Codes are not mutually exclusive. A single segment may carry multiple codes.
- Code families loosely correspond to emerging theme areas but final theme boundaries are determined in Phase 4.
- Example indicators are retrieval and indexing aids, not automatic proof that a segment supports a theme. Themes are developed through close reading, contextual interpretation, and memo-based reasoning.
- The `general_response` code captures substantive discussion content where keyword or pattern cues do not adequately index the segment. These segments remain analytically available and are interpreted through close reading, question context, speaker role, and memo-based reasoning during theme development.
- The `moderator_context` code is applied to all moderator turns. Moderator speech is retained for question-boundary detection and contextual understanding but is excluded from participant evidence, theme support, and quotation diversity.
- The `containment_emotional_holding` code captures the culturally specific concept of احتواء which emerged as a distinctive framing in Day 4 discussion — broader than empathy or reassurance, it describes a relational practice of making someone feel fully held and valued.
- The `age_dignity_language` code captures the deliberate preference for كبار القدر (people of standing/worth) over كبار السن (older people), which carries analytic weight as a cultural reframing of ageing.
