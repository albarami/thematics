# CASE_D1 Recommendation Workbook Usage Rule

## Source
`Day1_Childhood/Recommendations_Data/Health_Workshop_Suggestions Day 1.xlsx`

## Classification
Auxiliary structured recommendation material.

## Permitted uses
1. **Q7 practical suggestions** — The workbook directly contains participant-attributed practical suggestions. These may be cited in the Q7 analysis section and in recommendation tables as auxiliary evidence.
2. **Q6 opportunities** — Where suggestions imply available opportunities, the workbook may provide supplementary support for Q6 analysis.
3. **Recommendation tables** — Workbook content may populate structured recommendation summary tables in the final report.
4. **Policy/action implications** — Workbook suggestions may inform the policy implications discussion.

## Prohibited uses
1. **Theme generation** — Workbook content must not be used to generate, support, or evidence thematic claims.
2. **Transcript-equivalent evidence** — Workbook rows are not verbatim participant speech. They must never be treated as equivalent to transcript quotations.
3. **Quotation diversity** — Workbook "User" labels must not be counted toward participant quotation diversity metrics.
4. **Participant summary content** — Workbook rows must not appear in participant summary sheets as if they were transcript-derived evidence.
5. **Prominence/salience support** — Workbook content must not inflate theme prominence ratings.

## Labeling requirement
Wherever workbook content is used, it must be explicitly labeled as:
> *Source: auxiliary recommendation workbook (Health_Workshop_Suggestions Day 1.xlsx)*

This label must appear in the text, in tables, and in any structured output where workbook content is referenced.

## Workbook structure notes
- 40 suggestion rows (rows 2-41, including 2 supplementary rows at 40-41)
- Rows 40-41 have empty No/User fields but populated Suggestion (manually aggregated supplementary notes)
- Column structure: No | User | Suggestion
- "User" labels (User 1, User 2, etc.) are generic and cannot be linked to specific transcript participants
