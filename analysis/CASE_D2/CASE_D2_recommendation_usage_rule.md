# CASE_D2 Recommendation Workbook Usage Rule

## Scope
This note governs the use of `Day2_Youth/Recommendations_Data/Health_Workshop_Suggestions Day 2.xlsx` within CASE_D2 only.

## Classification
The Day 2 recommendation workbook is classified as **auxiliary structured recommendation material**.

It is **not** transcript-equivalent participant evidence.
It is **not** a theme-generating source.
It is **not** part of quotation diversity.

## Why this rule applies
The refreshed foundation assessment found that the workbook contains grouped recommendation rows with generic `User X` labels and no direct row-level linkage to transcript speakers or discussion questions.

## Permitted uses
- Clearly labeled support for `Q6` opportunity reporting
- Clearly labeled support for `Q7` practical-suggestion reporting
- Recommendation summary tables
- Policy/action implication tables or appendices

## Prohibited uses
- Generating themes
- Counting as participant evidence
- Counting as transcript evidence
- Counting toward quotation density or quotation diversity
- Being merged into the excerpt bank without explicit auxiliary labeling
- Being used to support question-to-theme claims unless separately triangulated with transcript evidence

## Labeling rule
Whenever workbook material is used in CASE_D2 outputs, it must be labeled as one of the following:
- `[auxiliary]`
- `auxiliary recommendation material`
- `recommendation workbook (not transcript-equivalent evidence)`
